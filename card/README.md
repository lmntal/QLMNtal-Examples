## card

### 中間命令列への直し方

### examples

- atoms
- link
- mem
  - [x] c(L1),c(L2),c(L3), {a(L1),a(L2),a(L3)}. {<1,>a(X),$p} :- {<1,>b(X),$p}.
    - -> c(L1),c(L2),c(L3), {b(L1),b(L2),b(L3)}.
    - 膜の中に a(X) 以外のプロセスがあっても、書き換える
  - [x] c(L1),c(L2),c(L3),c(L4), {a(L1),a(L2)}, {a(L3),b(L4)}. {<1,>a(X)} :- {<1,>b(X)}
    - c(L1),c(L2),c(L3),c(L4), {b(L1),b(L2)}, {a(L3),b(L4)}.
    - 膜の中に a(X) しかない場合にのみ、書き換える
- nest
  - [x] {a,a}, {a,a,a}, token. <1,>{<1,>a,$p}, token :- <1,>{<1,>b,$p}.
    - {b,b}, {b,b,b}.
    - 1個以上の a がある膜が1個以上あったとき、aをbに書き換える
