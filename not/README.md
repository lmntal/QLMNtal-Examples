## not

### 存在否定量化テンプレート

<^>

### 中間命令列への直し方

1. 書きたいnotを含んだQLMNtalのルールからnotを除いたルールを作る
2. notを含まないルールをコンパイルし、中間命令列を得る
3. 2. で得られた中間命令列において、否定したいマッチング部分を not[[]] で囲む
4. notで囲まれた部分にあるatomやmemは存在せず、remove/free する必要がないため、消去する

### examples

- atoms
  - [x] a. <^>a :- a. -> a.
  - [x] b. <^>a :- a. -> b, a.
  - [x] a, a. <^>a, a, a :- c, c. -> c, c.
  - [x] a, a. <^>(a, a, a) :- a. -> a, a, a.
- link
  - [x] a(X), b(X). c(Y), b(Y). <^>a(X) :- .
    - -> a(X), b(X). c(Y), b(Y).
  - [x] a(X), c(X). a(X), <^>b(X) :- a(X), b(X).
    - -> a(X), b(X).
- mem
  - [x] {a}, {b}. {<^>a, $p} :- {a, $p}.
    - -> {a}, {a, b}.
  - [x] {a, a}, {c}. <^>{a} :- {a}.
    - -> {a}, {c}, {a, a}.
  - [x] {b, b, b}, {a}. {<^>a, $p} :- {a, $p}.
    - -> {b, b, b, a}, {a}.
  - [x] {b, b, b}, {a, a, a}, {a}. {<^>a} :- .
    - {a}.
  - [x] {b, b}. <^>{a, $p} :- {a, $p}.
    - -> {b, b, a}.
  - [x] d(X), {b(X)}. <^>(a(X), {b(X)}) :- a(X), {b(X)}.
    - -> d(X), {b(X)}, a(Y), {b(Y)}.
- nest
  - [x] {a}, {a, b}, {a, a, b}. <^>{<^>a, $p} :- {$p}, {c}
    - -> {a}, {a, b}, {a, a, b}, {c}.
